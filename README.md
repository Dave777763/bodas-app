# 💍 Bodas App - Invitaciones Digitales Inteligentes

Una plataforma moderna diseñada para gestionar invitaciones de boda personalizadas, con temas dinámicos, confirmación RSVP en tiempo real y gestión de invitados.

## ✨ Características Principales

### 🎨 Sistema de Temas y Animaciones
Personalización total de la experiencia del invitado con 6 temas profesionales:
- **Romance Rosa** 🌹: Clásico y tierno.
- **Elegancia Azul** 💎: Sofisticado y formal.
- **Moderno Esmeralda** 🌿: Fresco y contemporáneo.
- **Rústico Terracota** 🍂: Cálido y vintage.
- **Tropical Coral** 🌺: Alegre y vibrante.
- **Minimalista Pizarra** ⚪: Limpio y moderno.

### 🎭 Animaciones Premium
- **Pétalos Cayendo**: Efectos visuales adaptables al color del tema.
- **Cuenta Regresiva**: Contador en tiempo real para el gran día.
- **Transiciones Suaves**: Animaciones de entrada (fade, scale, slide) optimizadas para GPU.

### 🛠️ Herramientas Administrativas
- **Dashboard en Tiempo Real**: Estadísticas de asistencia y visualización de recursos.
- **Gestión de Invitados**: Control de pases, nombres y estados de confirmación.
- **Check-in con QR**: Escáner integrado para la recepción del evento.
- **Integración Multimedia**: Soporte para música (Spotify/YouTube Music) y galería de fotos.

## 🚀 Inicio Rápido

### Prerrequisitos
- Node.js (v18+)
- Cuenta de Firebase

### Configuración
1. Clona el repositorio.
2. Ejecuta `setup_project.bat` en la raíz para configurar el entorno automáticamente.
3. Edita `frontend/.env.local` con tus credenciales de Firebase.

### Desarrollo
Para iniciar el servidor de desarrollo local:
```bash
./iniciar_app.bat
```
O manualmente:
```bash
cd frontend
npm run dev
```
